# Rosti
A crispy software rasteriser

## Goals
- Draw filled polygons and strokes
- Platform independent
- Fast
- No bloat & minimal dependencies
- Antialiassing
- Well documented API

## Non Goals
- Reading font files
- Drawing to screen
- Writing to image files
- GPU acceleration
